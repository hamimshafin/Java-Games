Java Mini Games Collection

This repository contains two classic mini games built using Java:

🏓 Ping Pong Game
🐍 Snake Game

These projects were created as part of my early learning journey in Java, following YouTube tutorials and adapting the code to better understand game development concepts.

🚀 Overview

This repository showcases fundamental programming concepts such as:

Object-Oriented Programming (OOP)
Game loops and real-time updates
Event handling (keyboard input)
Collision detection
Basic graphics rendering using Java
🏓 Ping Pong Game

A simple 2-player Ping Pong (Paddle) game.

🎯 Features
Two-player gameplay
Ball physics and collision detection
Score tracking system
Keyboard controls for both paddles
🎮 Controls
Player 1: W (Up), S (Down)
Player 2: ↑ (Up), ↓ (Down)
🐍 Snake Game

A classic Snake game where the player controls a snake to eat food and grow longer.

🎯 Features
Smooth snake movement
Food spawning system
Score tracking
Game over detection on collision
🎮 Controls
↑ Move Up
↓ Move Down
← Move Left
→ Move Right
🛠️ Technologies Used
Java
Java Swing / AWT (for GUI and rendering)
